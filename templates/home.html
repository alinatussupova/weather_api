Weather Data API

URL format: http://127.0.0.1:5000/api/v1/station/date

URL example for one specific station and date: http://127.0.0.1:5000/api/v1/10/1988-10-25

URL example for the station's all temperature changes for the entire period: http://127.0.0.1:5000/api/v1/10

URL example for one specific station and specific year: http://127.0.0.1:5000/api/v1/yearly/10/1988

Notes:

STAID: Station Identifier (ID)
